# String Algorithms

這個資料夾用來放與「字串處理、字串演算法」相關的程式練習與題目。

##  本資料夾內容
- 字串週期 (period)

##  檔案說明
### `period.cpp`
計算字串的週期（period）  
輸入字串，判斷是否可以由某一段重複組成，並求最小週期。

用途：
- 字串壓縮分析
- 迴圈字串檢查
- CPE 相關題型練習
